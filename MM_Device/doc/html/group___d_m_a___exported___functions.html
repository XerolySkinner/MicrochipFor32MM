参数
flagspecifies the flag to clear. This parameter can be any combination (for the same DMA) of the following values:
  • DMA1_FLAG_GLn: DMA1 Channeln global flag(n = 1..7).
  • DMA1_FLAG_TCn: DMA1 Channeln transfer complete flag(n = 1..7).
  • DMA1_FLAG_HTn: DMA1 Channeln half transfer flag(n = 1..7).
  • DMA1_FLAG_TEn: DMA1 Channeln transfer error flag(n = 1..7).
  • DMA2_FLAG_GLn: DMA1 Channeln global flag(n = 1..5).
  • DMA2_FLAG_TCn: DMA1 Channeln transfer complete flag(n = 1..5).
  • DMA2_FLAG_HTn: DMA1 Channeln half transfer flag(n = 1..5).
  • DMA2_FLAG_TEn: DMA1 Channeln transfer error flag(n = 1..5).
