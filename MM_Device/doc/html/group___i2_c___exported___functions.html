ErrorStatus I2C_CheckEvent ( I2C_TypeDef *  i2c,
u32  event 
)

Checks whether the last i2c Event is equal to the one passed as parameter.

参数
i2cselect the I2C peripheral.
eventspecifies the event to be checked. This parameter can be one of the following values:
  • I2C_EVENT_RX_UNDER : Rx Buffer is empty event
  • I2C_EVENT_RX_OVER : RX Buffer Overrun event
  • I2C_EVENTT_RX_FULL : Rx buffer full event
  • I2C_EVENT_TX_OVER : TX Buffer Overrun event
  • I2C_EVENT_TX_EMPTY : TX_FIFO empty event
  • I2C_EVENT_RD_REQ : I2C work as slave or master event
  • I2C_EVENT_TX_ABRT : TX error event(Master mode)
  • I2C_EVENT_RX_DONE : Master not ack event(slave mode)
  • I2C_EVENT_ACTIVITY : I2C activity event
  • I2C_EVENT_STOP_DET : stop condition event
  • I2C_EVENT_START_DET: start condition event
  • I2C_EVENT_GEN_CALL : a general call address and ack event
    • SUCCESS: Last event is equal to the I2C_EVENT
    • ERROR: Last event is different from the I2C_EVENT
