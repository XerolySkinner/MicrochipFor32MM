◆ TIM_PWMShiftConfig()

void TIM_PWMShiftConfig ( TIM_TypeDef *  tim,
u32  it,
FunctionalState  state 
)

Enables or disables the specified TIM PWM shift /DMA reqeat.

参数
timselect the tim peripheral.
itSpecifies the TIM PWM shift channel to enable or disable. This parameter can be any combination of the following values:
  • TIM_PDER_CCDREPE: TIM DMA reqeat enable bit
  • TIM_PDER_CCR1SHIFTEN: TIM Channel 1 output PWM phase shift enable bit
  • TIM_PDER_CCR2SHIFTEN: TIM Channel 2 output PWM phase shift enable bit
  • TIM_PDER_CCR3SHIFTEN: TIM Channel 3 output PWM phase shift enable bit
  • TIM_PDER_CCR4SHIFTEN: TIM Channel 4 output PWM phase shift enable bit
  • TIM_PDER_CCR5SHIFTEN: TIM Channel 5 output PWM phase shift enable bit
statenew state of the TIM interrupts. This parameter can be: ENABLE or DISABLE.
返回值
None.
