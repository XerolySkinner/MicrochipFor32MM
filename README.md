# **灵动32板级支持包**
----------------------------------------
## **🌸一个针对灵动32芯片的板级驱动包**
## **🍌为方便构建恩智浦杯智能车比赛的程序框架而诞生**
## **🍑基于灵动官方库**
----------------------------------------
## **功能**
  - 提供了一个空闲中断搭配DMA的与上位机通讯模块
  - 提供了一个基于官方库的快速初始化方案
  - 提供了一个软件IIC,并且可配置多路
  - 提供了一个用户配置User.h的模板
  - 提供了一颗草莓→🍓
